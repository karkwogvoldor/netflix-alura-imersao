# 🎬 Netflix Alura - Imersão

Projeto desenvolvido durante a Imersão Front-End da Alura. A aplicação consiste em uma interface responsiva inspirada na plataforma de streaming Netflix, focando em semântica HTML, estilização moderna com CSS e organização de layout.

## 🔗 Demonstração
> [clique aqui para acessar o link](https://karkwogvoldor.github.io/netflix-alura-imersao/)

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização, Flexbox, Grid e animações de hover.
*   **JavaScript (Opcional):** Manipulação de DOM para interatividade (se aplicado).
*   **Font Awesome/Google Fonts:** Tipografia e ícones.

## ⚙️ Funcionalidades

*   **Interface Responsiva:** Adaptável para diferentes tamanhos de tela (Desktop, Tablet e Mobile).
*   **Hover Effects:** Animações sutis ao passar o mouse sobre os cards dos filmes/séries.
*   **Menu de Navegação:** Cabeçalho fixo com transição de transparência.
*   **Banner Principal:** Destaque para o título principal com botões de interação.
